Math Minion is a general purpose calculation tool that runs completely in the browser and is completely free and private.

While it does have many high powered mathmatical tools, it can also be a more adept, transparent and powerful way of performing many of the tasks usually done in spreadsheets and smallish databases.

When things get more complicated, adding notes to the tools and grouping calculations into nested models can be helpful. Notes can also be displayed on the model view.

It combines a mind map style diagram of the relationships between a range of calculation tools, with a formula system that operates transparently on scalars, arrays and matrices. It also includes a comprehensive conversion unit system that performs dimensional analysis such that it knows dividing a btu/h value by a voltage results in amps, but adding a litre and a pound would be an error.

Best of all, if you have a reasonably recent browser and can wait a few seconds for it to load, you can try it right now by tapping on Run and check out the Getting Started session.

Known Problems

  • iPad Trackpad

    Pinching on the iPad trackpad to zoom the diagram zooms the entire window on Safari and Firefox, rather than just the diagram.

    It seems to work correctly on the Chrome and Edge browsers.

Math Minion and CoolProp

There is an older version of Math Minion available that has a fluid flash tool based on the open source CoolProp property package.

I have stopped updating this with the latest Math Minion features in the absence of any feedback indicating there interest in it, but you can still run it at coolmm.mathminion.org. Please be patient, for there is a lot of code to download. It has examples with fairly detailed explanations and of course help for the flash tool.

The source code is in the GitHub repository as branch coolmm.
