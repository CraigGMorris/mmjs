Math Minion is a general purpose calculation tool that runs completely in the browser and is completely free and private.

It combines a mind map style diagram of the relationships between a range of calculation tools, with a formula system that operates transparently on scalars, arrays and matrices. It also includes a comprehensive conversion unit system that performs dimensional analysis on all calculations.

Best of all, if you have a reasonably recent browser and can wait a few seconds for it to load, you can try it right now by tapping on Run.

Known Problems

  • Data Loss with Apple Safari!!!
    An update to Safari in late May 2021 introduced changes to its cross site tracking provisions that impact the in browser database that Math Minion uses to store sessions. It appears that this database gets cleared when the browser restarts and perhaps in other conditions.

    I think this occurs because I use a CName redirect to make mathminion.org/run.html point to the less friendly real address:

    https://s3.amazonaws.com/www.mathminion.org/run.html

    This affects Safari on the Mac and all browsers on iOS devices. Turning off cross site tracking in the preferences solves this or if you don't want to do that, just use the link above instead of the normal run link. Note that using a different link will start with just the Getting Started session, so be sure to export your sessions from the original link and reimport them with the new one.

    It is important to frequently export your sessions so you have hard drive backup of the browser based storage.

  • iPad Mouse and Trackpad
    There is some strange behaviour when you mix trackpad (or mouse) use with touch dragging or pinching on the screen. The latter will always work, but disables the trackpad on the diagram, graphs and result tables. Also pinch zooming on the trackpad doesn't work and should be avoided. Use the + and - zoom icons on the diagram instead. Reloading will restore things.

    Any thoughts on the cause of this would be most welcome.
