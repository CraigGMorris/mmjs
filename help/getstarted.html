Getting Started Session

The first time you run Math Minion, a Getting Started session ("session" is what I call a Math Minion document) will be automatically loaded.

It is is a heavily commented guide as well as a demonstration of Math Minion's features and tools. It is an excellent place to browse and try things out.

Don't worry about screwing it up, for you can always create a new copy by clicking on this link:

Note this will erase any changes you have made to the current Getting Started session

New Getting Started

If that doesn't work for some reason, try copying the command:

/ loadurl ../help/Getting Started.txt

and then clicking on the Console button at the bottom of any information view. Paste the command into the input field and hit return to create a fresh copy.
