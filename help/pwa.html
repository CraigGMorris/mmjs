Progressive Web Apps

Math Minion is configured as a progressive web app (PWA), which brings a number of benefits:

  • A PWA can be installed to operate as a standalone app. It still runs in the browser, but in a clean window of its own and with an application icon that you can use to launch it, just like a regular app.
  • All of its source is cached on your device, which means it loads quickly and can be used offline.
  • If the server source code is updated, the cached code will automatically be updated when the app is run and the device is online.

Installing as an App

While most of the benefits of being a PWA are realized even if you don't install it as an app, you may appreciate the cleaner interface and ability to launch from an icon. How you install will depend on the browser and device you use, but here is of some I have tried.

Desktop Chomium Browsers
These browsers, including Google Chrome, MS Edge, Brave and others, have a small icon at the right of the address bar when a web app can be installed. Hover over it and a tool tip saying something like "App available. Install Math Minion". Just click on it and follow the instructions.

Safari on a Mac
While running Math Minion in the browser, click on the share icon on the top bar and select Add to Dock. This will place a Math Minion icon on the dock and clicking on it will launch a standalone version of Math Minion. As far as I know, there is no way to have this icon anywhere else.

iOS and iPadOS Devices
While running Math Minion in a browser, tap on the share icon and scroll down and select Add to Home Screen. This will place a Math Minion icon on the home screen. It can be moved or put in a folder like any other app icon.

Android
I don't have an Android device to try this on, but it sounds like, depending on browser and device, a button saying something like Install or Add to Home Screen will appear.

If you have information that would improve these notes, please let me know.

Help in Standalone App

Note that when you tap on one of Math Minion's help buttons in the standalone app the help opens in the browser in a new tab or window in the browser, rather than in a new app window.
