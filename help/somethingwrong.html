That's not Good

If you get the message that something went wrong, then poor Math Minion is afflicted with a bug in its page rendering. Hopefully you will never see this, it is a bug and I definitely want to hear about it. Please let me know so I can fix it.

First though, let's try and get you running again. Try the following steps:

  • Try just reloading the page. If the error doesn't immediately reappear, try and see if you can reproduce it and let me know.
  • If the message reappears as soon as the program reloads, but you can see the diagram, try tapping on a different tool icon on the diagram and then reload again. If the bug is with a specific tool, this will probably work. You may need to remove the problem tool and add it back. Again let me know.
  • Try the URL https://mathminion.org/run.html?cmd=/%20new. This should launch Math Minion with a new session. From there you can access the Session page and try loading a different session. If this works, try exporting the problem session to a file and contact me. I may be able to recover the session and hopefully fix whatever went wrong.
  • If none of that works, contact me and I will try to help

Never Ending Calculaton

I don't think (??) there is a possibility of an actual infinite loop, but even so it is possible to create a problem whose resolution might take an unacceptable amount of time. While the user interface will remain responsive, the calculation thread will ignore messages from it in such circumstances. Even worse, in some cases the calculation may just start again even if you refresh the browser or quit and relaunch the app.

If this happens, switch to the browser if you are using the app and use this URL:

https://mathminion.org/run.html?cmd=/%20new

which should launch Math Minion with a new session. At this point, use the Console button at the bottom of the information view to open the console. Then select Console from the drop down menu at the bottom and enter:

/ set noRun t

Note: case counts.

This will disable the iterative tools, namely:

  • Equation Solver
  • Optimizer
  • ODE Solver
  • Iterator

Use the Console button again to toggle off the console and load the problem session and try to reconfigure the problem appropriately. Then return to the console and enter:

/ set noRun f

to reenable the tools.

Important

Please be sure and regularly do an Export All from the Sessions Page. This could be vital for recovery if all else fails!
