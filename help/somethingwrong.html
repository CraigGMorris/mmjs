That's not Good

If you get the message that something went wrong, then poor Math Minion is afflicted with a bug in its page rendering. Hopefully you will never see this, but if you do, I definitely want to hear about it.

First though, let's try and get you running again. Try the following steps:

  • Try just reloading the page. If the error doesn't immediately reappear, try and see if you can reproduce it and let me know.
  • If the message reappears as soon as the program reloads, but you can see the diagram, try tapping on a different tool icon on the diagram and then reload again. If the bug is with a specific tool, this will probably work. You may need to remove the problem tool and add it back. Again let me know.
  • Try the URL https://mathminion.org/run.html?cmd=/%20new. This should launch Math Minion with a new session. From there you can access the Session page and try loading a different session. If this works, try exporting the problem session to a file and contact me. I may be able to recover the session and hopefully fix whatever went wrong.
  • If none of that works, contact me and I will try to help

Important

Please be sure and regularly do an Export All from the Sessions Page. This could be vital for recovery if all else fails!
