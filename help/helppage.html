Help Pages

At the top of all help pages is a non-scrolling header with the following buttons

  • < - Previous help page if applicable
  • Title - tapping will scroll back to the top of the page
  • > - Next help page (if you have gone to a previous one)
  • X - closes the help window or tab

Below the header are the Section Menu and Help Contents buttons. Tapping on Section Menu will expand to a set of links for quickly jumping to a section of the current page. Tapping on Help Contents will reveal a list of links to all the available help pages.

If you are new to Math Minion, be sure to check out the Getting Started Session, Introductory Tutorial, and the Navigate Video.

Information View Help Button

At the top right of the information view there is a context sensitive help button that will display help for the current view.
