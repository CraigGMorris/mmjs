New News

Rather than just posting news on the main web page, where it probably would be missed by folks actually running Math Minion, I am trying something new.

Now when Math Minion loads and there is new news, it will automatically download and open a MM_News session from the server. The idea behind using a session rather than a simple text post is that it may sometimes be helpful to include examples along with the text. In any event the HtmlPage tool provides an easy way to present information.

Feel free to delete the MM_News session. It won't be downloaded again until there is more news. Please note that if you make any changes to the MM_News session, they will be replaced the next time there is news, so save them somewhere else if you want to keep them.

Server Address Changes (Again)

I previously posted some information on the main web page about problems that arose with a recent update of Apple's Safari web browser.

In essence they decided that any address using a CName DNS record for redirection must be trying to avoid their anti tracking measures and so they clear that site's browser database whenever the browser quits. In Math Minion's case this would have the effect of erasing all saved sessions. Not good!

The CName record was being used to redirect from mathminion.org the the Amazon AWS S3 bucket that acts as a static server for Math Minion. I initially suggested using the actual, if ugly, Amazon url to run Math Minion, namely:

https://s3.amazonaws.com/
www.mathminion.org/run.html

This required exporting the sessions from the existing address and importing them for the new address. Browser databases are only accessible to the domain names that create them, so an address change means a new database.

Unfortunately using the Amazon address has its own problems, but by using an Amazon DNS server, it is possible to avoid using the CName record Safari gets sniffy about. So I am now recommending going back to:

http://mathminion.org/run.html

Alas this will mean once again moving your sessions. Fortunately it is pretty easy:

  • Use the Amazon address to run Math Minion. Tap the Sessions button at the bottom of the information view to open the sessions view and click on the Export All button. Save the exported file someplace convenient. (You should do this frequently to ensure you have a backup of your sessions.
  • Now use the http://mathminion.org/run.html address to run Math Minion.
  • If you have any old sessions saved from before, you should probably delete them to avoid duplicates.
  • Go to the sessions view again and click on import. Select the file you saved.
  • The imported sessions will be in a top level folder called root. Go to the sessions view and click on the gear icon to the right of the root folder name.
  • Select Rename and rename /root to just /. This will have the effect of moving all your session to the top level folder.

    Note that if a session already exists at the top level with the same name as one in /root, the file from /root will be put in a folder named copy-2 or similar.

My apologies for all the trouble.
