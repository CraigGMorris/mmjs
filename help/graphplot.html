Graph Plot

When you tap the Plot button on the graph object view, the data is graphically presented in this view. Tap the back arrow to return to the graph definition view.

If a Z value was defined in the graph, a three dimensional (3D) plot will appear, otherwise it will be a standard two dimensional (2D) x-y plot.

Plot Axes

Both 2D and 3D plots can have multiple X value arrays, with associated Y and, for 3D, Z value arrays. A 2D plot can have more than one set of Y values associated with each X array, while a 3D plot only has one set of Y and one set of Z values for each X array.

When a plot has more than one set of X or Y values, the axis labels will only represent one at a time. If there is more than one set of X values, a single tap on the X axis label will switch to the next set or cycle back to the first if the last one is currently displayed. The X and Y, and Z for 3D, axis labels will change to reflect the new set.

When the current X axis for a 2D plot has more than one Y value, tapping on any Y axis label will switch the Y axis labels to the values of the next Y line.

For a 2D plot, the colour of the Y axis labels will always match the plotted values (lines or dots) that they represent and the X axis label will be blue if there is more than one set of X values.

On a 3D plot, all the axis labels will be the same colour as the plotted data they represent and the presence of more than one colour of plotted data indicates multiple X value sets.

Zooming

To zoom in and out on the graph, use the standard pinch gesture where you move your thumb and a finger together to zoom in and apart to zoom out. The axis labels will adjust their values appropriately. If you are on the iPad, you may want to switch to full screen mode (button with outward pointing arrows).

2D Panning

Simply drag the graph around with one finger to pan it back and forth and up and down. The axis labels will adjust their values appropriately.

3D Panning

Dragging horizontally at the lower right of the view will move the plotted data back and forth along the X axis, while dragging horizontally at the lower left will move the image along the Y axis. Dragging vertically along the left edge of the view, starting about mid height of the view, will move the image vertically along the Z axis.

In all cases, the axis labels will adjust their values appropriately.

3D Rotation

The image, including the axis frame, can be rotated around the Z axis by dragging horizontally along the top of the view, starting from about mid width.

Rescale & Center

Double tap on the plot to have it return to its original scaling and position.

Export Image

Tap on the gear icon on the top tool bar to export the graph image by printing, emailing or opening it in another app as a pdf file or to place the image in your camera roll. The latter can be used to place the image in a document or spreadsheet.
