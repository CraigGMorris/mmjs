Overview

Every Math Minion tool can have notes associated with it. Beside the name field at the top of every tool information view there is a button labeled Add Notes or Show Notes if notes have already been added. Tapping that will open the notes view.

You can enter any text you want into the large text area of the notes view. Tapping on the Apply / Hide button beside the name field will assign the changed notes to he tool and return to the normal information view.

If you switch to another tool by tapping on the diagram without tapping either the Apply / Hide button or the Show on Diagram button, any changes you have made will be lost.

You can access the notes for any tool in a formula with the notes parameter. For example the formula:

velcocity.notes

would return a string value containing the notes for tool velocity.

Diagram Notes

If you wish to have your notes appear on the diagram, tap on the button labeled Show on Diagram. The notes will appear on the diagram below the tool icon and the button will change to Hide on Diagram as in the image above.

Note that if the note is very long, it might be truncated on the diagram.

If a tool has notes and they are not shown on the diagram, there will still be the word Notes below the icon to indicate the tool has hidden notes.
