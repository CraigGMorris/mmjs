Getting Help from ChatGPT

I have created a couple of ChatGPT assistants for Math Minion. These are chat sessions that start with specialized knowledge for a given task. You should think of them as somewhat dim interns rather than as experts. They can save you time for simple tasks, but they will make mistakes and their work should be checked carefully.

If you have the ChatGPT app installed, clicking on an assistant link should open it with the assistant ready to follow your instructions. If you don't have the app, you should be taken to a new ChatGPT web page, again with the assistant ready.

Tools Assistant

Describe your problem to ChatGPT and it will attempt to create a JSON representation of a suitable Math Minion tool, usually a Model, to solve your problem. Copy and paste the result onto the Math Minion diagram to create the actual tool.

The results may require some fixing, but usually provide a good starting point and can be much faster than starting from scratch.

Data Table Assistant

Give this assistant some representation of a table, text or image and it will attempt to create a Math Minion CSV file appropriate for pasting on the diagram to create a data table tool.
