Export

This dialog appears when you ask to export a plot or diagram image or the text from a report or string expression. The options are:

eMail

An image will be attached as a pdf file to an email in a composition window. Text will be attached as a txt file.

Print

The information is printed using Air Print.

Open

A list of applications that can open the image or text is presented.

Image

This only appears for images and if selected it copies the image to the camera roll. Inserting images from the camera roll is how many application add images to documents, spreadsheets etc.
